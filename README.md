# ProjectCSA2001
# AI-Based Expense Analyzer

## 📌 Project Overview

This project is a simple AI-based expense tracking system built using Python. It helps users record their daily expenses, analyze spending patterns, and generate basic AI-based insights such as overspending warnings and future spending predictions.

The project focuses on applying basic concepts of data analysis and simple AI logic using real-life data.

---

## 🎯 Problem Statement

Managing daily expenses is difficult, especially for students. Many people do not track where their money goes, which leads to overspending.

This project solves this problem by:

* Recording expenses
* Categorizing spending
* Providing analysis and suggestions

---

## ⚙️ Features

* Add daily expenses (Amount, Category, Date)
* View total and category-wise spending
* Time-based analysis (daily, weekly, monthly)
* AI-based suggestions (overspending detection)
* AI-generated report with insights

---

## 🛠️ Technologies Used

* Python
* Pandas (for data handling)
* CSV file (for storing data)

## ▶️ How to Run the Project

### Step 1: Install Requirements

Make sure Python is installed. Then install pandas:

```
pip install pandas
```

### Step 2: Run the Program

Open terminal in project folder and run:

```
python main.py
```
